Styling

At first I had to style the whole page according to comments located in html file. I've styled UI cards almost the same way as I did it in the first exercise and used the same colors as in the second exercise. To layout the UI cards on the website I used grid with responsive columns (using repeat(auto-fill, minmax())). To make other parts of the website responsive, I used media queries. I've changed all flexes in row direction to column and fixed some margins and paddings in many different elements. Moreover I used responsive units like rem or vw.
